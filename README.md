## Summary:

* Projects for __An Introduction to Interactive Programming in Python__ by Rice University.

* To run the project game: just  copy/paste code and run on [CodeSkulptor](http://www.codeskulptor.org), or click the link list below for each game. 
	* CodeSkulptor runs Python programs in your browser.  
	* CodeSkulptor runs in Chrome 18+, Firefox 11+, and Safari 6+.  
	* Some features may work in other browsers, but do not expect full functionality.  
	* It does NOT run in Internet Explorer.
